Présentation de SQL

SQL (sigle pour Structured Query Language, « langage de requêtes structurées ») est un langage informatique normalisé servant à exploiter des bases de données relationnelles. La partie langage de manipulation des données de SQL permet de rechercher, d'ajouter, de modifier ou de supprimer des données dans les bases de données relationnelles.

Mon avis sur cette technologie

  • Manipulation très stricte et rigoureuse, comparé à du noSQL, plus souple et facilement modifiable
  • Optimale pour la gestion de tableaux, très fortement utilisé dans le secteur bancaire/assurantielle
  • Sécurité et fiabilité idéale, mais également très performant pour la gestion de grands volumes de données
